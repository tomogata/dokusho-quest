# 読書クエスト

読書記録をRPG風に楽しむWebアプリです。

🌐 **公開URL**: https://dokusho-quest.netlify.app

---

## 使い方

### 1. ステータスを確認する
起動すると最初に「ステータス」タブが表示されます。レベル・EXP・5つのパラメータ・現在の職業を確認できます。

| パラメータ | 主なジャンル |
|-----------|------------|
| 知識力 | 人文・学術・ビジネス・科学 |
| 感受力 | 文芸・詩・恋愛・エッセイ・マンガ・コミック |
| 探究力 | ミステリー・SF |
| 世界力 | 歴史・地理・旅行 |
| 創造力 | ファンタジー・芸術・マンガ・コミック |

### 2. 本を登録する
1. 「どくしょきろく」タブを開く
2. タイトルを入力して「けんさく」ボタンを押す
3. 検索結果から読んだ本を選ぶ
4. ジャンル・ページ数・読了日を確認して「とうろくする」をクリック

> ジャンルが違う場合は手動で変更してから登録してください

### 3. 称号を集める
「しょうごう図鑑」タブで獲得できる称号を確認できます。
冊数・ページ数・ジャンルなど22種類の称号があります。

### 4. 職業について
読んだ本のジャンルの組み合わせで職業が変わります。

| レア度 | 説明 |
|--------|------|
| つうじょう | 最初から目指せる職業 |
| レア | 2つのパラメータが突出すると出現 |
| ちょうレア | 3つ以上のパラメータが高水準で出現 |
| でんせつ | 特定の高水準条件を満たした読書家のみ |

> 特定の冊数（10・20・30・40・50・60・75・100冊）の節目でサプライズ職業が出現することも！

### 5. Xに投稿する
レベルアップや職業変化のポップアップで「𝕏 に投稿する」ボタンを押すと、テンプレート文章入りの投稿画面が開きます。ポップアップのスクリーンショットを添付して投稿しましょう。

### 6. きろくをほぞん・よみこむ
読書履歴の右上にある「↓ ほぞん」ボタンでデータをJSONファイルとして書き出せます。
「↑ よみこむ」ボタンで保存したファイルを読み込めます。

> キャッシュクリアや別の端末で使う前に「↓ ほぞん」でバックアップを取っておきましょう。

## アプリの概要

本を読むたびに経験値（EXP）が貯まり、レベルアップしていきます。
読んだ本のジャンルに応じてパラメータが成長し、職業が変化します。

### 主な機能

- 読んだ本のタイトル・著者・ページ数・ジャンルを登録
- Google Books APIによる本の情報自動取得（intitle:検索で精度向上）
- レベル・EXP・5つのパラメータ（知識力・感受力・探究力・世界力・創造力）
- 34種類の職業（通常9・レア16・超レア9・伝説3・ネタ8）
- 称号図鑑（22種類）
- 読書記録のエクスポート／インポート機能（JSONファイル）
- 読了時にXへ投稿する機能
- Google Analyticsによるアクセス解析

### 職業一覧（34種類）

**通常（9種類）**
学徒、司書、詩人、探偵、旅人、夢想家、吟遊詩人、民俗学者、ロマン主義者

**レア以上は読書を続けて自分で発見してください👀**
レア・超レア・伝説・サプライズ職業など、まだまだ隠された職業が存在します。
どんな読書をすれば出会えるのか…？随時追加予定です。

### 称号一覧（22種類）

**冊数系（6種類）**
はじめの一歩、本の友、十冊読者、三十冊の求道者、五十冊の探求者、百冊読者

**ページ数系（5種類）**
超人強度の読書家、千ページの旅人、五千ページの賢者、一万ページの覇者、五万ページの伝説

**ジャンル系（4種類）**
ジャンル探検家、万能読書家、ボールは友達、全ジャンル制覇

**特殊（7種類）**
長編の覇者、速読の使徒、全集中の読書家、年間読書家、小宇宙の覚醒者、限界を超えた者、諦めない者

## 既知の問題・メモ

### ジャンルの自動判定精度
Google Books APIから取得するジャンル情報の精度が高くない場合があります。
特に翻訳書や日本語の本はカテゴリが的外れになることがあります。

**対処法**: 検索後の入力フォームでジャンルを手動で修正してから登録してください。

### データの保存について
現状はブラウザのlocalStorageに保存しています。
別のブラウザや端末ではデータが引き継がれません。

**対処法**: 「↓ ほぞん」でJSONファイルをエクスポートしてバックアップを取ってください。将来的にはクラウド保存対応予定。

### 検索機能について
Claude.aiのアーティファクト内ではAnthropicのAPIで検索しています。
Netlifyで公開されたバージョンはGoogle Books APIを使用しています。
