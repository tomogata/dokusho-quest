# 読書クエスト

読書記録をRPG風に楽しむWebアプリです。

🌐 **公開URL**: https://dokusho-quest.netlify.app

---

## アプリの概要

本を読むたびに経験値（EXP）が貯まり、レベルアップしていきます。
読んだ本のジャンルに応じてパラメータが成長し、職業が変化します。

### 主な機能

- 読んだ本のタイトル・著者・ページ数・ジャンルを登録
- Google Books APIによる本の情報自動取得
- レベル・EXP・5つのパラメータ（知識力・感受力・探究力・世界力・創造力）
- 25種類の職業（通常・レア・超レア・伝説）
- ネタ職業（10冊・20冊・30冊・50冊・75冊・100冊の節目で出現）
- 称号図鑑（16種類）
- ドラゴンクエスト3パッケージ風デザイン

---

## 既知の問題・メモ

### ジャンルの自動判定精度
Google Books APIから取得するジャンル情報の精度が高くない場合があります。
特に翻訳書や日本語の本はカテゴリが的外れになることがあります。

**対処法**: 検索後の入力フォームでジャンルを手動で修正してから登録してください。

### データの保存について
現状はブラウザのlocalStorageに保存しています。
別のブラウザや端末ではデータが引き継がれません。

**今後の対応予定**: エクスポート／インポート機能の追加、将来的にはクラウド保存対応。

### 検索機能について
Claude.aiのアーティファクト内ではAnthropicのAPIで検索しています。
Netlifyで公開されたバージョンはGoogle Books APIを使用しています。
